# Stoik Architecture

## Purpose

Stoik is a personal production assistant for Nuke artists.
Its purpose is to automate repetitive production tasks.

---

# Folder Structure

core/
Business logic.

ui/
Qt interface.

utils/
Utility functions.

templates/
Nuke templates.

icons/
Icons.
